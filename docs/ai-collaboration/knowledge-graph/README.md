## 🎯 两个版本的区别

### MVP版（mvp-data.json）
**适用场景**：游戏MVP版本、原型开发、快速验证

**包含内容**：
- 仅南山经·鹊山系（10座山）
- 新手区域全部内容
- 核心异兽、植物、矿物
- 鹊山山神祭祀系统
- 难度分级标注
- MVP玩法指引

**设计原则**：
- 小而精，足够支撑5-10小时游戏体验
- 每个实体都有明确的游戏用途
- 覆盖生存、医药、祭祀三大基础科技树

## 🎮 MVP版本玩法设计

### 新手流程
1. **出生点**：招摇之山（难度1）
   - 采集祝馀（食之不饥）
   - 佩戴迷穀（不迷路）
   - 狩猎狌狌（食之善走）

2. **探索区域**：堂庭之山 → 杻阳之山（难度1-2）
   - 收集水玉、黄金
   - 发现鹿蜀（佩之宜子孙）
   - 遇到旋龟（佩之不聋）

3. **挑战区域**：青丘之山（难度4，Boss山）
   - 九尾狐（史诗级异兽）
   - 灌灌（佩之不惑）
   - 赤鱬（食之不疥）

### 科技树解锁
- **生存**：采集、狩猎、建造
- **医药**：草药、解毒、治疗
- **祭祀**：鹊山山神祭祀、祈福

## 🔧 如何使用

### 其他AI分支接入
```javascript
// 读取MVP版本（推荐，稳定）
const mvpData = require('./mvp-data.json');

// 读取完整版（内容更多，但可能更新频繁）
const fullData = require('./full-data.json');
```

### 数据查询示例
```javascript
// 查询某座山有哪些异兽
const mountain = fullData.mountains.find(m => m.id === 'mountain_qingqiu');
console.log(mountain.creatures); // ['creature_jiuwei', 'creature_guanguan', 'creature_chiru']

// 查询九尾狐的详细信息
const creature = fullData.creatures.find(c => c.id === 'creature_jiuwei');
console.log(creature.effects.eat); // '食者不蛊'
```

所有数据均严格遵循以下原则：
