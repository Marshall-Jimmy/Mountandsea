# 文言文风格指南与命名规范

**版本**: 0.1.0  
**最后更新**: 2026-06-20  
**维护者**: 文化内核AI负责人  
**作用**: 确保游戏中文言文内容风格统一，文化质感一致

---

## 一、为什么需要风格统一？

六个AI各写各的文言文，最后会出现：
- 有的像先秦散文，有的像明清小说
- 有的用"之乎者也"堆砌，有的太白话
- 同一个东西，这个AI叫"丹药"，那个AI叫"丹丸"

玩家一看就出戏，文化质感全没了。

---

## 二、总体风格定位

### 文风基准
**以《山海经》原文为基准**，参考先秦诸子散文风格：
- 简洁凝练，不堆砌辞藻
- 多用四字句、短句
- 少用复杂的修辞手法
- 客观叙述，少主观评价

### 难度分级
游戏提供三档文言文难度：

| 难度 | 风格 | 适用玩家 |
|------|------|---------|
| 白话 | 现代白话文，通俗易懂 | 普通玩家 |
| 浅白 | 半文半白，有古风但易懂 | 大部分玩家 |
| 先秦 | 接近山海经原文风格 | 硬核玩家/文化爱好者 |

**默认使用「浅白」档**，同时提供三档文本供玩家选择。

## 三、命名规范

### 3.1 物品命名

**原则**：
1. 优先使用山海经/典籍中的原名
2. 没有原名的，按"材质/功能 + 类别"的方式命名
3. 避免现代词汇

**正面示例**：
- ✅ 祝馀（山海经原名）
- ✅ 迷穀（山海经原名）
- ✅ 玉璋（祭祀用玉器）
- ✅ 草药（通用类别）

**反面示例**：
- ❌ 生命药水（太现代）
- ❌ 经验丹（游戏术语）
- ❌ 超级治疗药（太直白）

### 3.2 技能/科技命名

**原则**：
1. 用动词开头，如"采"、"炼"、"祭"
2. 两到四字为佳
3. 符合典籍中的说法

**正面示例**：
- ✅ 采集
- ✅ 炼丹
- ✅ 筑基
- ✅ 御剑术

**反面示例**：
- ❌ 采集术（"术"字多余）
- ❌ 炼丹技能（太直白）
- ❌ 伤害加深（游戏术语）

### 3.3 地名命名

**原则**：
1. 优先使用山海经中的山名、地名
2. 原创地名要符合"某山"、"某水"、"某之野"的格式
3. 避免现代地名

**正面示例**：
- ✅ 招摇之山
- ✅ 青丘之野
- ✅ 西海

**反面示例**：
- ❌ 新手村（太游戏化）
- ❌ 北京上海（现代地名）

### 3.4 NPC命名

**原则**：
1. 用先秦风格的名字
2. 姓氏+名，或直接称"某父"、"某母"
3. 避免太现代的名字

**正面示例**：
- ✅ 巫咸
- ✅ 岐伯
- ✅ 老父

**反面示例**：
- ❌ 王小明（太现代）
- ❌ 村长（太游戏化）

## 四、文案写作规范

### 4.1 物品描述

**格式**：
```
[物品名]
[一句话外形描述]
[效果/用途]
```

**示例（浅白档）**：
```
祝馀
状如韭而青华
食之不饥
```

**示例（先秦档）**：
```
祝馀
其状如韭而青华
食之不饥
```

### 4.2 科技描述

**格式**：
```
[科技名]
[一句话说明这是什么]
[解锁内容/效果]
```

**示例**：
```
炼丹
以草药金石炼取丹药
解锁：解毒丹、不死药
```

### 4.3 剧情文本

**原则**：
- 多用对话，少用叙述
- 语言符合人物身份
- 避免大段独白

**示例**：
```
老父捋须笑道：
"后生欲入山？山中多异兽，需小心。"
```

## 五、禁用词汇清单

### 绝对禁用（现代游戏术语）
- ❌ 经验、等级、升级
- ❌ HP、MP、血量、蓝量
- ❌ 装备、背包、技能栏
- ❌ 任务、奖励、成就
- ❌ 伤害、防御、攻击力
- ❌ NPC、BOSS、小怪
- ❌ 玩家、账号、存档

### 尽量避免（后世词汇）
- ⚠️ 佛、菩萨、和尚（佛教东汉传入）
- ⚠️ 道士、道观（道教后起）
- ⚠️ 纸（汉代发明）
- ⚠️ 茶（唐代盛行）
- ⚠️ 椅子（唐代以后）

### 替代方案

| 现代词 | 文言替代 |
|--------|---------|
| 经验 | 修为、道行 |
| 等级 | 境界、层次 |
| HP/血量 | 气血、性命 |
| 装备 | 随身、披挂 |
| 背包 | 行囊、包裹 |
| 任务 | 差事、托付 |
| 奖励 | 酬劳、赏赐 |
| 伤害 | 伤势、损耗 |
| 防御 | 护体、防身 |

## 七、AI写作检查清单

每个AI生成文言文内容后，请自查：

- [ ] 有没有现代游戏术语？
- [ ] 有没有明显的后世词汇？
- [ ] 风格是否和山海经原文接近？
- [ ] 命名是否符合规范？
- [ ] 有没有和知识图谱中的设定矛盾？
- [ ] 三档文本是否都提供了？
