# 山海经附录贰 · 科技树

> 基于山海经游戏设定，设计从原始到文明的科技发展路线。
> 作者：杨泽宇
> 设计理念：科技树分为六大分支，玩家可选择不同发展路线。

## 一、生存科技

> 从最基本的生存技能开始，逐步掌握驾驭自然的能力。

### 1.1 采集
- **解锁条件**：起始
- **效果**：可采集野生草药、果实
- **可采集**：祝余、萆荔、薲草等基础草药

### 1.2 狩猎
- **解锁条件**：采集
- **效果**：可狩猎普通野兽获取肉食与皮毛
- **工具**：石矛、木棒

### 1.3 生火
- **解锁条件**：采集
- **效果**：可烹饪食物，驱赶野兽
- **烹饪解锁**：熟祝余、熟鯥鱼等熟食

### 1.4 渔猎
- **解锁条件**：狩猎
- **效果**：可捕鱼获取鱼类资源
- **可捕获**：滑鱼、儵鱼、何罗鱼等

### 1.5 建造
- **解锁条件**：生火
- **效果**：可建造简易住所
- **建筑**：茅屋、木棚、石屋

### 1.6 制皮
- **解锁条件**：狩猎
- **效果**：可加工兽皮制作装备
- **可制作**：兽皮衣、溪边皮席（防蛊）

### 1.7 驯兽
- **解锁条件**：狩猎 + 农牧（畜牧）
- **效果**：可驯服部分异兽
- **可驯服**：鹿蜀、水马、吉量（需高级驯兽）

## 二、医药科技

> 基于山海经中丰富的草药与异兽效果，发展医疗体系。

### 2.1 草药学
- **解锁条件**：采集
- **效果**：识别并使用基础草药
- **可识别**：祝余（不饥）、条草（不惑）、薰草（已疠）

### 2.2 烹药
- **解锁条件**：草药学 + 生火
- **效果**：可将草药熬制成药汤
- **可制作**：
  - 萆荔汤（治心痛）
  - 黄雚浴汤（治疥疮、足肿）
  - 杜衡汤（治瘿瘤）

### 2.3 解毒术
- **解锁条件**：草药学
- **效果**：可制作各类解毒药
- **可制作**：
  - 蝮虫毒牙解药
  - 百毒清（需耳鼠，御百毒）

### 2.4 外科手术（西医中世纪）
- **解锁条件**：烹药 + 工匠（金属加工）
- **效果**：基于四液理论的外科手术
- **阶段一（四液理论）**：
  - 放血疗法
  - 灌肠术
  - 草药敷料
- **阶段二（推翻四液，需科技达标）**：
  - 解剖学
  - 消毒术
  - 缝合术

### 2.5 炼丹
- **解锁条件**：烹药 + 修仙（引气）
- **效果**：可炼制丹药
- **可炼制**：
  - 白䓘膏（不饥、释劳）
  - 玉膏（高级恢复）
  - 不死药（需不死树，传说级）

## 三、修仙科技

> 基于山海经神话体系的修炼路线。

### 3.1 引气
- **解锁条件**：祭祀（信仰）或 自悟（随机）
- **效果**：感知天地灵气，开始修仙之路
- **简介**：修仙的起点，需在有灵气之地修炼

### 3.2 筑基
- **解锁条件**：引气（熟练度达标）
- **效果**：构筑修仙根基
- **辅助物品**：玉膏、白䓘膏

### 3.3 御物
- **解锁条件**：筑基
- **效果**：可操控物体
- **应用**：御剑飞行（需高级）、隔空取物

### 3.4 御兽
- **解锁条件**：筑基 + 驯兽
- **效果**：可驯服并驱使异兽
- **可御使**：九尾狐、狡、吉量等

### 3.5 飞升
- **解锁条件**：修仙满级 + 特殊事件
- **效果**：飞升成仙
- **简介**：最终目标，需完成特定神话事件链

## 四、祭祀科技

> 2022.11.3 新增系统，基于山海经中的山神祭祀礼仪。

### 4.1 基础祭祀
- **解锁条件**：生火
- **效果**：可进行简单的祭祀仪式
- **祭品**：毛（牲畜毛皮）、糈（祭米）

### 4.2 山神祭祀
- **解锁条件**：基础祭祀
- **效果**：可祭祀各山山神，获得庇佑
- **祭祀规则**（依原文）：

| 山系 | 神状 | 祭品 |
|------|------|------|
| 南山经 | 鸟身龙首 | 毛用一璋玉瘗，糈用稌米，一璧，稻米白菅为席 |
| 南次二经 | 龙身鸟首 | 毛用一璧瘗，糈用稌 |
| 南次三经 | 龙身人面 | 一白狗祈，糈用稌 |
| 西山经 | — | 太牢（华山冢）、百牺百瑜（羭山神） |
| 北山经 | 人面蛇身 | 毛用一雄鸡彘瘗，吉玉用一圭 |
| 中山经 | — | 各山不同 |

### 4.3 通灵术
- **解锁条件**：山神祭祀（熟练度达标）
- **效果**：可与神灵沟通
- **可沟通**：山神、巫咸、灵山十巫

### 4.4 祈愿术
- **解锁条件**：通灵术
- **效果**：向神灵祈愿
- **可祈愿**：
  - 风调雨顺（防旱防涝）
  - 百兽不侵
  - 五谷丰登
  - 疾病消散

## 五、工匠科技

> 从石器到金属加工，再到机械制造。

### 5.1 石器
- **解锁条件**：起始
- **效果**：可制作石器工具
- **可制作**：石刀、石斧、石矛、磨石

### 5.2 木工
- **解锁条件**：石器
- **效果**：可加工木材
- **可制作**：木棒、木弓、木屋、木车（需高级）

### 5.3 陶器
- **解锁条件**：生火
- **效果**：可制作陶器
- **可制作**：陶罐、陶碗、陶鼎（用于烹药）

### 5.4 金属加工
- **解锁条件**：陶器 + 生火（高级）
- **效果**：可冶炼金属
- **可冶炼**：铜、金、银、铁
- **可制作**：铜刀、铜鼎、金属工具

### 5.5 玉器加工
- **解锁条件**：石器 + 金属加工
- **效果**：可加工玉石
- **可制作**：璋、璧、圭（祭祀用玉器）
- **特殊**：白䓘可血玉（染玉作光彩）

### 5.6 机械
- **解锁条件**：木工 + 金属加工
- **效果**：可制作机械装置
- **可制作**：
  - 陷阱（捕兽）
  - 飞车（参考奇肱国，需高级）
  - 弩机

## 六、农牧科技

> 从采集狩猎到定居农耕。

### 6.1 种植
- **解锁条件**：采集
- **效果**：可种植作物
- **可种植**：稌（稻）、黍、稷

### 6.2 灌溉
- **解锁条件**：种植 + 建造
- **效果**：可修建灌溉系统
- **建筑**：水渠、水车

### 6.3 畜牧
- **解锁条件**：驯兽
- **效果**：可饲养牲畜
- **可饲养**：牛、羊、猪、马

### 6.4 育种
- **解锁条件**：种植 + 畜牧
- **效果**：可改良品种
- **特殊**：鹿蜀角（佩之宜子孙，可增强后代天赋）

### 6.5 百谷**
- **解锁条件**：种植（高级）
- **效果**：可种植多种作物
- **简介**：参考后稷降以百谷的传说

## 特殊科技

### 息壤
- **解锁条件**：海内经事件链
- **效果**：可获得自动生长的土壤，用于填堵洪水
- **简介**：鲧窃帝之息壤以堙洪水。传说级物品。

### 彤弓素矰
- **解锁条件**：羿事件链
- **效果**：神弓，可射杀异兽
- **简介**：帝俊赐羿彤弓素矰，以扶下地之百艰。

### 夔牛鼓
- **解锁条件**：大荒东事件链 + 工匠（高级）
- **效果**：声闻五百里，可震慑群兽
- **简介**：以夔牛之皮为鼓，其声如雷。

> **设计备注**：
> 1. 科技树允许跨分支组合，鼓励玩家探索不同发展路线
> 2. 部分高级科技需要触发特定神话事件链
> 3. 西医外科手术的两阶段设计体现了"科技推翻旧理论"的核心理念
> 4. 祭祀系统的祭品规则严格遵循山海经原文设定
