## 一、设计理念与核心矛盾

### 1.1 传统RPG激励循环的局限

传统RPG的激励循环可以概括为：

```
任务指派 → 完成任务 → 获得奖励（经验/装备/金币）→ 数值提升 → 更难的任务
```

这个循环有效但有三个致命问题：

1. **奖励疲劳**：玩家很快对数值提升麻木，「+10攻击力」不再令人兴奋
2. **难度通胀**：数值线性增长，难度必须同步通胀，否则失去挑战——最终变成纯粹的数字比大小
3. **外在动机挤出内在动机**：一旦玩家习惯了「做事=拿奖励」，任何没有奖励的事都会被跳过，包括叙事、探索、沉浸

### 1.2 本项目的核心矛盾

山海经项目做了一个激进的设计选择：**剧情任务没有奖励**。这意味着：

- 不能靠「任务→奖励→新任务」循环驱动玩家
- 不能靠数值膨胀制造虚假的进步感
- 不能靠稀有装备掉落维持长期兴趣

矛盾在于：**没有外在奖励，玩家为什么要玩下去？**

答案在于重新定义「奖励」。在本项目中，奖励不是数值，而是**可能性**——新的交互方式、新的知识、新的生存策略。玩家玩下去的动力不是「变强」，而是「变懂」。

### 1.3 设计原则

基于以上分析，本激励方案遵循五条设计原则：

| 原则 | 含义 | 对应层级 |
|------|------|---------|
| 知识永久 | 知识一旦获得不会因死亡丢失，是真正的「永久进度」 | 第一层 |
| 解锁即负债 | 每一项科技解锁都同时引入新的问题和负担 | 第二层 |
| 压力周期化 | 灾变不是随机惩罚，而是可预见可准备的周期性事件 | 第三层 |
| 死亡即进步 | 死亡不是失败，而是传承的触发点，为下一代铺路 | 第四层 |
| 稀缺驱动探索 | 关键资源地理独占，远征是获取传说级科技的唯一途径 | 第五层 |

## 二、第一层：知识即权力

### 2.1 山海经作为「攻略本」

山海经原文本身就是一部「地理志怪」——它记载了每座山的物产、异兽、植物、矿物，以及它们的功效。这在游戏语境下就是一本**攻略本**。

设计核心：**把山海经原文变成游戏内的图鉴系统，玩家初始状态下一无所知，必须通过探索和试错逐步揭开世界真相。**

玩家初次进入招摇山时，看到的只是一片有植物和动物的陌生地形。他不知道：
- 那棵韭菜一样的草叫「祝余」，吃了可以不饥
- 那只白耳猿猴叫「狌狌」，吃了可以加快移动速度
- 那棵发光的树叫「迷榖」，佩戴可以在雾中辨方向
- 水里有蝮虫，被咬了会中毒，需要解药

这些信息全部来自山海经原文，但玩家需要**自己去发现**。

### 2.2 图鉴系统设计

#### 图鉴条目结构

每个可交互实体（异兽/草木/矿物/神灵）在图鉴中有一个条目，条目分为多个**信息槽位**，初始全部锁定：

```
┌─────────────────────────────────────────┐
│  祝余                                     │
│  ─────────────────────────────────────   │
│  [外观]  其状如韭而青华                    │
│  [类型]  草                               │
│  [效果]  食之不饥                          │
│  [烹饪]  熟祝余：食之不饥（时长×4）         │
│  [产地]  招摇山                            │
│  [稀有度] 普遍                             │
│  [备注]  气味有韭菜味，可致饱腹感            │
│                                          │
│  解锁进度：■■■□□□□ (3/7)                 │
└─────────────────────────────────────────┘
```

#### 解锁条件

| 槽位 | 解锁方式 | 说明 |
|------|---------|------|
| 外观 | 第一次目视接触 | 走到该实体附近自动触发 |
| 类型 | 第一次目视接触 | 与外观同时解锁 |
| 效果 | 第一次亲眼目睹/亲身经历 | 如：看到NPC吃祝余不饿，或自己吃了发现不饿 |
| 烹饪 | 第一次成功烹饪 | 需要先解锁效果+拥有生火科技 |
| 产地 | 第一次在该地点采集到 | 记录具体山系 |
| 稀有度 | 采集/狩猎累计达10次 | 统计后判定 |
| 备注 | 特殊事件触发 | 如被蝮虫咬后解毒，解锁蝮虫备注 |

#### 图鉴的激励效应

- **完成欲**：人类天生有收集癖，未解锁的槽位会驱动玩家反复尝试
- **实用性**：解锁的效果信息直接指导玩家行为——知道「食之不饥」才会去吃
- **叙事性**：备注槽位往往关联一个小故事，提供碎片化叙事
- **传承价值**：图鉴是跨代永久的，每一代的探索都在丰富家族的知识库

### 2.3 试错与信息不对称

#### 信息不对称的三种类型

| 类型 | 例子 | 难度来源 |
|------|------|---------|
| 属性未知 | 九尾狐「能食人」 | 玩家不知道它会攻击人，第一次接近可能被吃 |
| 用途未知 | 蝮虫毒牙可做解药 | 玩家不知道毒牙有用，可能直接丢弃 |
| 条件未知 | 祭祀需要特定祭品 | 祭品错误会触怒山神 |

#### 试错的安全阀

试错机制如果过于残酷会导致玩家放弃。设计以下安全阀：

1. **濒死警告**：玩家血量低于20%时，视野边缘出现红色脉动，提示撤退
2. **传承预览**：死亡前可以看到「传承预览」——后代将继承哪些知识、哪些天赋，让玩家做出有信息的死亡决策
3. **知识碎片**：即使试错失败，也能获得部分信息。如被九尾狐攻击但逃脱，图鉴中九尾狐的「效果」槽位会显示「能食人（疑似）」，标注为推测
4. **NPC口传**：村落中的NPC会提供部分图鉴信息作为对话内容，降低完全盲目的试错

#### 信息不对称的难度梯度

```
完全无知（初始）
    ↓ 第一次接触
推测信息（图鉴标注「疑似」）
    ↓ 多次接触或目睹
确认信息（图鉴标注「已知」）
    ↓ 深度交互
精通信息（图鉴标注「精通」，解锁高级用法）
```

### 2.4 知识的传承与永久性

#### 传承规则

| 信息类型 | 死亡后是否保留 | 说明 |
|---------|--------------|------|
| 图鉴条目 | ✅ 保留 | 所有已解锁的槽位永久保留 |
| 地图探索 | ✅ 保留 | 已探索区域的地形保留，但异兽分布可能变化 |
| 科技解锁 | ✅ 保留 | 已解锁的科技节点不因死亡丢失 |
| 配方记忆 | ✅ 保留 | 已学会的配方（烹药/炼丹/锻造）保留 |
| 技能熟练度 | ❌ 不保留 | 每一代从零开始练 |
| 物品装备 | ❌ 不保留 | 随角色死亡消失（除非有仓库） |

#### 知识作为「真正的进度」

这个设计的关键在于：**玩家感受到的「进度」不是角色的数值，而是自己作为玩家对世界的理解**。

第一代玩家可能花了5小时学会「祝余充饥」「蝮虫有毒」「生火烹饪」。这5小时的成果不会因角色死亡而白费——第二代玩家（后代）出生时就已经知道这些。第二代可以花这5小时去探索更远的地方、学习更高级的知识。

这就是「知识即权力」——不是角色的权力，是**玩家的权力**。

### 2.5 知识解锁的交互可能性

知识不只是图鉴里的文字，它直接解锁新的**交互选项**：

| 知识 | 解锁前 | 解锁后 |
|------|--------|--------|
| 祝余「食之不饥」 | 看到一棵草，无法交互 | 可以采集祝余，食用充饥 |
| 蝮虫毒牙可做解药 | 毒牙是垃圾，直接丢弃 | 可以收集毒牙，制作蝮毒解药 |
| 迷榖「佩之不迷」 | 看到一棵发光的树，不知道有什么用 | 可以采集迷榖，制作防迷路饰品 |
| 鹿蜀角「佩之宜子孙」 | 鹿蜀角是普通掉落物 | 可以制作饰品，佩戴后提升后代天赋 |
| 山神祭祀规则 | 祭祀是随机操作 | 可以按正确祭品祭祀，获得山神庇佑 |

**设计要点**：知识解锁的不是「更强的攻击力」，而是「新的游戏玩法」。这保证了难度不会因知识积累而消失——你知道九尾狐能吃人，但你仍然需要准备防蛊装备、规划逃跑路线、选择合适的武器。

## 三、第二层：科技树的「解锁即负债」

### 3.1 传统科技树的问题

传统科技树的设计是「前置→解锁→纯增益」。玩家解锁金属加工后获得铜刀（攻击力+10）、铜鼎（炼药效率+50%），一切都是正向的。

问题在于：**纯增益意味着玩家只会越来越强，难度必然下降**。为了维持难度，游戏只能不断推出更强的敌人——这就是「数值通胀」的根源。

### 3.2 负债设计原则

「解锁即负债」的核心是：**每一项科技解锁都同时引入新的问题和负担**。

负债不是惩罚，而是**新的游戏内容**。玩家解锁金属加工后，确实获得了铜刀，但也需要：
- 伐木烧炭（燃料消耗）
- 采矿（劳动投入）
- 维护工具（铜器会磨损）
- 面对生态后果（森林砍伐导致异兽迁徙）

这些不是「减益buff」，而是**新的玩法维度**。玩家从「采集者」变成了「管理者」——管理资源、管理生态、管理风险。

### 3.3 六大科技树负债表

#### 生存科技负债

| 科技 | 增益 | 负债 | 生态反馈 |
|------|------|------|---------|
| 采集 | 可采集草药果实 | 无 | 无 |
| 狩猎 | 可获取肉食皮毛 | 需要武器（消耗石器/木工） | 过度狩猎导致该区域异兽减少 |
| 生火 | 可烹饪、驱兽 | 需要燃料（木材） | 频繁生火可能引发野火 |
| 渔猎 | 可捕鱼 | 需要渔具 | 过度捕捞导致鱼类资源枯竭 |
| 建造 | 可建住所 | 需要大量木材 | 伐木导致森林减少 |
| 制皮 | 可制作皮甲 | 需要兽皮+鞣制剂 | 制皮过程产生异味，吸引某些异兽 |
| 驯兽 | 可驯服异兽 | 需要持续喂食 | 被驯服的异兽可能引来野生同类攻击 |

#### 医药科技负债

| 科技 | 增益 | 负债 | 风险 |
|------|------|------|------|
| 草药学 | 识别基础草药 | 需要记忆草药形态 | 误认草药可能导致中毒 |
| 烹药 | 可熬制药汤 | 需要陶鼎+燃料+药材 | 煎煮时间/火候错误降低药效 |
| 解毒术 | 可制作解毒药 | 需要特定材料（如耳鼠） | 部分解毒药本身有副作用 |
| 外科手术·阶段一 | 放血/灌肠/敷料 | 需要金属工具 | 误诊加速死亡，四液理论可能错误 |
| 外科手术·阶段二 | 解剖/消毒/缝合 | 需要高级科技支撑 | 需大量实践，初期失败率高 |
| 炼丹 | 可炼制高级丹药 | 需要丹炉+稀有材料 | 丹炉爆炸风险，失败消耗材料 |

#### 修仙科技负债

| 科技 | 增益 | 负债 | 风险 |
|------|------|------|------|
| 引气 | 感知灵气 | 需在有灵气之地修炼 | 离开灵气地修为停滞 |
| 筑基 | 构筑修仙根基 | 需要辅助物品（玉膏/白䓘膏） | 筑基失败可能走火入魔 |
| 御物 | 可操控物体 | 持续消耗灵力 | 灵力耗尽导致虚弱 |
| 御兽 | 可驱使异兽 | 需要驯兽前置+灵力维持 | 被御使的异兽可能反噬 |
| 飞升 | 飞升成仙 | 需完成特定神话事件链 | 飞升失败=角色永久消失（无传承） |

#### 祭祀科技负债

| 科技 | 增益 | 负债 | 风险 |
|------|------|------|------|
| 基础祭祀 | 简单祭祀仪式 | 需要祭品（毛/糈） | 祭品不足无效 |
| 山神祭祀 | 获得山神庇佑 | 需要特定祭品（璋/璧/圭/稌米） | 祭品错误触怒山神降灾 |
| 通灵术 | 可与神灵沟通 | 需要高熟练度+特定法器 | 通灵失败可能被邪灵附身 |
| 祈愿术 | 可向神灵祈愿 | 需要大量祭品+通灵前置 | 祈愿被拒会降低神灵好感度 |

#### 工匠科技负债

| 科技 | 增益 | 负债 | 生态反馈 |
|------|------|------|---------|
| 石器 | 石刀/石斧/石矛 | 石器易碎，需频繁制作 | 采石破坏地表 |
| 木工 | 木弓/木屋/木车 | 需要大量木材 | 伐木导致森林减少 |
| 陶器 | 陶罐/陶碗/陶鼎 | 需要黏土+窑+燃料 | 烧窑消耗大量木材 |
| 金属加工 | 铜刀/铜鼎/金属工具 | 需要矿石+炭+高温窑 | 炼铜产生废气，冶炼区草木枯萎 |
| 玉器加工 | 璋/璧/圭（祭祀用） | 需要玉石+金属工具 | 玉石矿稀少，需远征 |
| 机械 | 陷阱/弩机/飞车 | 需要木工+金属加工 | 机械需要维护，故障可能伤己 |

#### 农牧科技负债

| 科技 | 增益 | 负债 | 生态反馈 |
|------|------|------|---------|
| 种植 | 可种作物 | 需要种子+土地+时间 | 单一种植导致土壤肥力下降 |
| 灌溉 | 水渠/水车 | 需要建造+维护 | 改变水文，影响下游生态 |
| 畜牧 | 牛/羊/猪/马 | 需要饲料+围栏+看护 | 牲畜密集可能引发瘟疫 |
| 育种 | 改良品种 | 需要多代选育+知识 | 近亲繁殖导致退化 |
| 百谷 | 多种作物 | 需要高级种植+灌溉 | 管理复杂度高 |

### 3.4 生态反馈系统

科技负债的核心载体是**生态反馈系统**——玩家的行为会改变环境，环境变化会反过来影响玩家的生存条件。

#### 生态指标

每个区域（chunk）维护以下生态指标：

| 指标 | 范围 | 影响 |
|------|------|------|
| 森林覆盖率 | 0-100 | 低于30时异兽迁徙、木材短缺、水土流失 |
| 水质 | 0-100 | 低于50时饮水有概率致病，鱼类减少 |
| 土壤肥力 | 0-100 | 低于40时作物减产，需要施肥 |
| 异兽密度 | 0-100 | 过高威胁村落，过低导致资源短缺 |
| 灵气浓度 | 0-100 | 低于20时无法修仙，高于80吸引神灵 |

#### 生态反馈链

```
伐木烧炭（金属加工燃料）
    ↓
森林覆盖率下降
    ↓
水土流失 → 土壤肥力下降 → 农作物减产
    ↓
异兽失去栖息地 → 迁徙至村落附近 → 袭击事件增加
    ↓
玩家需要更多武器防御 → 更多金属加工 → 更多伐木
    ↓
恶性循环
```

**打破循环的方式**：
- 选择性伐木（只伐特定树种，保留森林骨架）
- 种植速生树种（农牧科技支撑）
- 迁移村落（放弃已有基建，代价巨大）
- 祭祀祈愿「百兽不侵」（临时缓解）

### 3.5 科技倒退与文明崩溃

在极端情况下（连续灾变、生态崩溃、血脉断绝），科技可以**倒退**：

- 工坊被毁且工匠死亡 → 该科技节点暂时不可用，需重新建造工坊
- 唯一掌握某配方的角色死亡且未传承 → 配方丢失（但图鉴记录保留，可重新研发）
- 村落被异兽摧毁 → 所有基建归零，但科技树解锁状态保留

**设计意图**：科技不是一条只能前进的单行道。文明的脆弱性本身就是难度来源——你建起的一切都可能失去，但你的知识和图鉴永远不会丢。这创造了「重建」的叙事动力。

## 四、第三层：环境压力循环

### 4.1 预兆型异兽系统

山海经原文中有大量**预兆型异兽**——它们出现就预示灾变。这是天然的难度周期化机制。

#### 预兆异兽完整表

| 异兽 | 出处 | 预兆 | 游戏内灾变 | 预警时间 |
|------|------|------|-----------|---------|
| 长右 | 南山经 | 郡县大水 | 洪水淹没低地 | 出现后3-5天 |
| 顒 | 南山经 | 天下大旱 | 干旱，水源枯竭 | 出现后5-7天 |
| 鱄鱼 | 南山经 | 天下大旱 | 长期干旱 | 出现后5-7天 |
| 肥𧔥 | 西山经 | 天下大旱 | 干旱+高温 | 出现后3天 |
| 毕方 | 西山经 | 火灾 | 野火蔓延 | 出现后1-3天 |
| 蛮蛮 | 西山经 | 天下大水 | 洪水 | 出现后3-5天 |
| 胜遇 | 西山经 | 其国大水 | 洪水 | 出现后3-5天 |
| 狸力 | 南山经 | 其县多土功 | 强制徭役事件 | 出现后7天 |
| 鴸 | 南山经 | 其县多放士 | NPC流放事件 | 出现后7天 |
| 猾褢 | 南山经 | 县有大繇 | 大型劳役事件 | 出现后7天 |
| 山𤟤 | 北山经 | 天下大风 | 风暴 | 出现后1-2天 |

### 4.2 灾变类型与机制

#### 洪水

| 阶段 | 持续时间 | 效果 | 应对 |
|------|---------|------|------|
| 预兆期 | 3-5天 | 长右出现，河流水位上升 | 向高地迁移，储存食物 |
| 爆发期 | 2-3天 | 低地淹没，建筑损坏，作物毁坏 | 在高处避难 |
| 退水期 | 3-5天 | 水位下降，地面泥泞，疾病风险 | 清理淤泥，消毒 |
| 恢复期 | 7-14天 | 土壤肥力提升（洪水带来淤泥） | 重新种植，修缮建筑 |

**难度递增**：洪水强度与玩家基建规模正相关。住茅屋的原始玩家损失小（茅屋好重建），有灌溉水车和大片农田的文明玩家损失大。

#### 干旱

| 阶段 | 持续时间 | 效果 | 应对 |
|------|---------|------|------|
| 预兆期 | 5-7天 | 顒/鱄鱼出现，溪流变细 | 储水，减少灌溉 |
| 爆发期 | 10-20天 | 水源枯竭，作物枯死，异兽为争水接近村落 | 节水，狩猎来水源饮水的异兽 |
| 恢复期 | 5-10天 | 降雨恢复，但部分作物已死 | 补种，调整种植策略 |

**应对手段**：
- 祭祀祈愿「风调雨顺」（需高级祭祀科技）
- 建蓄水池（需建造+灌溉科技）
- 种植耐旱作物（需育种科技）

#### 野火

| 阶段 | 持续时间 | 效果 | 应对 |
|------|---------|------|------|
| 预兆期 | 1-3天 | 毕方出现，空气干燥 | 清理建筑周围枯枝，挖防火带 |
| 爆发期 | 1-2天 | 火势蔓延，建筑/森林被焚 | 撤离，放弃建筑 |
| 恢复期 | 10-20天 | 焦土，但草木灰提升土壤肥力 | 重新种植，利用肥力加成 |

**特殊机制**：野火后烧焦的区域土壤肥力+30，是建立新农田的好时机。这体现了「灾变也是机遇」的设计理念。

#### 风暴

| 阶段 | 持续时间 | 效果 | 应对 |
|------|---------|------|------|
| 预兆期 | 1-2天 | 山𤟤出现，风力增强 | 加固建筑，收起户外物品 |
| 爆发期 | 1天 | 建筑损坏，飞行类异兽活跃，视野受限 | 室内避难 |
| 恢复期 | 2-3天 | 风停，部分树木倒伏 | 清理路障，修复建筑 |

### 4.3 灾变强度递增曲线

灾变强度不随时间线性增长，而是**与玩家科技水平挂钩**：

```
灾变强度 = 基础强度 × (1 + 科技等级系数) × (1 + 村落规模系数)

科技等级系数 = 已解锁科技节点数 × 0.05
村落规模系数 = 建筑数量 × 0.02
```

| 玩家阶段 | 科技节点 | 建筑 | 灾变强度倍率 |
|---------|---------|------|------------|
| 蛮荒期 | 0-3 | 0-2 | 1.0× |
| 定居期 | 4-8 | 3-8 | 1.3-1.6× |
| 文明期 | 9-15 | 9-20 | 1.7-2.3× |
| 神话期 | 16+ | 20+ | 2.5×+ |

**设计意图**：科技越发达，灾变破坏力越大。这不是惩罚强者，而是**让难度始终与能力匹配**。你有了灌溉水车，洪水就更有破坏力；你有了大片农田，干旱就更致命。

### 4.4 准备-灾变-恢复循环

```
    ┌──→ 观察预兆 ──→ 准备 ──→ 灾变爆发 ──→ 恢复重建 ──┐
    │                                                    │
    │              知识积累：灾变经验                     │
    │                                                    │
    └────────────────────────────────────────────────────┘
```

每次灾变后，玩家获得：
- 该灾变的图鉴条目（预兆异兽→灾变类型的因果关系）
- 灾变应对经验（下次预兆出现时反应更快）
- 生态变化数据（如洪水后土壤肥力提升）

**循环的激励效应**：玩家不是在「等待灾变结束」，而是在「利用灾变周期」。有经验的玩家会：
- 在预兆期低价收购他人放弃的物资
- 在灾变期狩猎因灾变被迫迁移的异兽
- 在恢复期利用生态变化（如洪水后的高肥力土地）抢种作物

### 4.5 季节与节气系统

灾变之外，季节变化提供**基础环境压力**：

| 季节 | 持续 | 环境效果 | 生存压力 |
|------|------|---------|---------|
| 春 | 15天 | 万物复苏，异兽繁殖期 | 幼兽出没，母兽攻击性增强 |
| 夏 | 15天 | 高温，水源充足 | 食物易腐，疾病多发 |
| 秋 | 15天 | 丰收，异兽囤积食物 | 异兽为储备冬粮而活跃 |
| 冬 | 15天 | 低温，部分植物枯萎 | 食物短缺，需储备过冬 |

**季节与灾变的叠加**：夏季干旱比秋季干旱更致命（高温+缺水），冬季洪水比春季洪水更危险（低温+潮湿）。

## 五、第四层：死亡传承的「基因赌局」

### 5.1 传承机制核心规则

当玩家角色死亡时，触发传承流程：

```
角色死亡
    ↓
传承预览（显示后代将继承的内容）
    ↓
选择后代（从已有子女中选，或生成新后代）
    ↓
后代出生/接替
    ↓
知识保留，熟练度重置，物品丢失
```

### 5.2 天赋遗传与变异

#### 天赋系统

每个角色有5项天赋，每项0-10级：

| 天赋 | 影响 |
|------|------|
| 体魄 | 生命值、负重、体力恢复 |
| 灵巧 | 攻击速度、采集效率、制作精度 |
| 悟性 | 学习速度、修仙效率、配方研发 |
| 感知 | 侦查范围、预兆识别、灵气感知 |
| 魅力 | NPC好感度、交易价格、驯兽效率 |

#### 遗传规则

```
后代天赋 = floor((父天赋 + 母天赋) / 2) + 随机变异

随机变异：
  50%概率：±0（无变异）
  30%概率：+1（正向变异）
  15%概率：-1（负向变异）
  5%概率：  +2（罕见正向变异）
```

**设计意图**：后代天赋不是简单的「取最高」或「平均值」，而是一个**赌局**。你可能生出天赋超群的后代（正向变异），也可能生出不如父母的后代（负向变异）。这创造了「血脉优化」的长期目标——玩家会刻意选择高天赋的配偶，佩戴鹿蜀角提高变异概率。

### 5.3 鹿蜀角的赌注设计

鹿蜀角「佩之宜子孙」是传承系统的核心赌注道具：

| 状态 | 效果 | 代价 |
|------|------|------|
| 不佩戴 | 正常遗传 | 无 |
| 佩戴鹿蜀角 | 正向变异概率+15%，后代天赋上限+1 | 占用一个饰品槽，当代战斗力下降 |

**赌注设计**：
- 佩戴鹿蜀角意味着当代少一个饰品槽——战斗/生存能力下降
- 但后代可能更强——长期收益
- 如果当代因战斗力不足而早死——后代还没出生就亏了
- 如果活到自然死亡——后代天赋提升，长期赚了

这就是「基因赌局」——你用当代的生存概率换后代的潜力。

### 5.4 尸体肥料与土地传承

任何生物的尸体都可以用作**肥料**：

| 尸体类型 | 肥料效果 | 特殊效果 |
|---------|---------|---------|
| 普通野兽 | 土壤肥力+5 | 无 |
| 异兽 | 土壤肥力+10 | 有概率赋予土地特殊属性 |
| 修仙者（玩家死亡） | 土壤肥力+20 | 有概率在死亡地点生长出特殊植物 |
| 神灵 | 土壤肥力+50 | 极高概率生长传说级植物 |

**土地传承**：玩家死亡后选择在死亡地点（或居住地）埋葬尸体，该地块获得永久肥力加成。下一代在该地块种植作物产量更高。这创造了「祖坟风水」的概念——你的历代先祖的埋葬地是最肥沃的土地。

**设计意图**：死亡不是纯粹的损失。你的角色死了，但他的尸体滋养了土地，后代在那片土地上活得更好。每一代人的死亡都在为下一代铺路。

### 5.5 知识与熟练度的分离

| 进度类型 | 死亡后 | 说明 |
|---------|--------|------|
| 图鉴知识 | ✅ 保留 | 永久进度，跨代积累 |
| 科技解锁 | ✅ 保留 | 文明层面的进步 |
| 配方记忆 | ✅ 保留 | 学会的制作配方 |
| 地图探索 | ✅ 保留 | 已探索区域 |
| 技能熟练度 | ❌ 重置 | 每代从零练起 |
| 物品装备 | ❌ 丢失 | 随角色消失 |

**设计意图**：知识和熟练度的分离是难度控制的关键。知识保留让玩家感到「每一代都没有白玩」，熟练度重置防止了无限滚雪球。

第三代玩家出生时：
- 已知祝余充饥、蝮虫有毒、如何生火（知识保留）
- 但采集速度、狩猎精度、烹饪技巧都是新手水平（熟练度重置）
- 需要花时间重新练熟练度，但不需要重新「发现」这些知识

### 5.6 血脉断绝条件

游戏结束（永久死亡）的条件：

1. **无后代死亡**：角色死亡时没有子女，血脉断绝
2. **后代全灭**：所有子女先于角色死亡
3. **飞升失败**：修仙飞升失败导致角色永久消失（特殊机制）

**防止血脉断绝的策略**：
- 尽早生育（成年后即可）
- 生育多个子女（分散风险）
- 保持村落安全（祭祀祈愿「百兽不侵」）
- 不冒不必要的险（但这样会错过很多内容）

**设计意图**：血脉断绝是最终的「Game Over」，但它不是突然发生的——玩家有充分的机会预防。这创造了「冒险vs保守」的持续张力：你想去远征获取稀有资源，但远征有死亡风险；你不去远征，科技停滞，最终灾变可能毁灭村落。

## 六、第五层：稀缺性设计与远征

### 6.1 地理独占资源

山海经原文中很多关键资源是**地理独占**的——只存在于特定山系。这是远征系统的文本基础。

#### 地理独占资源表

| 资源 | 产地 | 用途 | 距离参考 |
|------|------|------|---------|
| 玉膏 | 峚山 | 筑基辅助、高级恢复 | 需翻越西山数座山 |
| 不死树 | 昆仑之虚 | 不死药（传说级） | 极远，需穿越大荒 |
| 白䓘 | 仑者山 | 白䓘膏（不饥释劳）、血玉 | 西山深处 |
| 夔牛 | 大荒东 | 夔牛鼓（声闻五百里） | 极远，需渡海 |
| 息壤 | 海内经 | 堙洪水（传说级） | 需触发鲧事件链 |
| 彤弓素矰 | 海内 | 神弓（射杀异兽） | 需触发羿事件链 |
| 吉量 | 海内北 | 乘之寿千岁 | 需深入海内北 |
| 沙棠 | 昆仑丘 | 御水（食用不溺） | 西山深处 |
| 视肉 | 昆仑之虚 | 无限食物 | 极远 |
| 灵山十巫 | 大荒西 | 交易药物、治疗 | 极远，需穿越大荒 |

### 6.2 远征系统设计

#### 远征准备阶段

| 准备项 | 必需/可选 | 说明 |
|--------|----------|------|
| 食物 | 必需 | 按远征天数×2准备（来回） |
| 饮水 | 必需 | 或携带水囊，或规划沿途水源 |
| 武器 | 必需 | 防御沿途异兽 |
| 药品 | 推荐 | 解毒药、疗伤药 |
| 向导 | 可选 | 雇佣NPC向导（需高魅力） |
| 地图 | 可选 | 已探索区域的地图（知识保留） |
| 备用装备 | 推荐 | 武器会磨损，衣物会损坏 |

#### 远征过程

远征是一次**高风险的连续生存挑战**：

1. **行进**：每天消耗食物和水，可能遇到随机事件
2. **露营**：夜间需要生火驱兽，可能被夜行异兽袭击
3. **导航**：依靠地图知识（已探索区域）或试错（未探索区域）
4. **遭遇**：沿途异兽可能攻击、可能中立、可能提供信息
5. **资源点**：沿途可能有可采集的资源，但采集消耗时间
6. **目的地**：到达后需要找到资源、采集/狩猎，可能还需与占据资源的异兽战斗

#### 远征随机事件表

| 事件 | 概率 | 效果 |
|------|------|------|
| 遇到中立异兽 | 20% | 可选择绕行或狩猎 |
| 遇到攻击性异兽 | 15% | 战斗或逃跑 |
| 发现可采集资源 | 15% | 获得沿途资源 |
| 恶劣天气 | 10% | 行进速度减半，消耗增加 |
| 遇到NPC/村落 | 10% | 可交易、休息、获取信息 |
| 迷路 | 10% | 消耗额外时间，可能进入危险区域 |
| 发现遗迹/洞穴 | 5% | 可能获得稀有物品 |
| 无事发生 | 15% | 正常行进 |

### 6.3 传说级科技解锁

远征带回的稀有材料可以解锁**传说级科技**——这些是改变游戏体验的质变奖励：

| 传说科技 | 所需材料 | 效果 | 来源 |
|---------|---------|------|------|
| 息壤 | 触发鲧事件链 | 自动生长的土壤，可堙洪水 | 海内经 |
| 彤弓素矰 | 触发羿事件链 | 神弓，可射杀传说级异兽 | 海内 |
| 夔牛鼓 | 夔牛皮+工匠高级 | 声闻五百里，震慑群兽 | 大荒东 |
| 不死药 | 不死树+炼丹高级 | 长生不死（但有代价） | 昆仑之虚 |
| 飞车 | 奇肱国技术+机械高级 | 可飞行，快速移动 | 大荒西 |

**设计要点**：传说级科技不是「数值+100」，而是**新的游戏机制**。息壤可以永久解决洪水威胁（但需要触发鲧事件链，本身是一个长线任务）；彤弓素矰可以射杀 otherwise 无法击败的传说异兽；飞车可以快速远征（但需要燃料和维护）。

### 6.4 远征的风险与回报

#### 风险

- **角色死亡**：远征途中死亡，如果无同伴则物品全丢
- **物资消耗**：即使成功，食物/药品/武器损耗巨大
- **时间成本**：远征期间村落无人防守，可能遭灾
- **信息泄露**：远征路线可能被其他势力获知（如果有竞争势力）

#### 回报

- **稀有材料**：地理独占资源，无法通过其他途径获得
- **图鉴扩充**：沿途遇到的新异兽/植物自动录入图鉴
- **地图探索**：永久解锁沿途地图
- **传说科技**：质变性能力提升
- **叙事内容**：远征途中可能触发神话事件链

**设计意图**：远征是「all in」式的冒险——你投入大量资源和一个角色的生命，可能满载而归，也可能血本无归。这种高风险高回报的设计让玩家在「安稳发展」和「冒险突破」之间持续纠结。

## 七、难度曲线设计

### 7.1 整体难度曲线

```
难度
  ↑
  │                                          ╱─── 神话期（远征+传说科技）
  │                                        ╱
  │                          ╱─── 文明期（灾变+生态管理）
  │                        ╱
  │            ╱─── 定居期（科技负债+季节压力）
  │          ╱
  │  ╱─── 蛮荒期（信息不对称+生存压力）
  │╱
  └──────────────────────────────────────────→ 游戏时间
```

难度不是线性上升，而是**阶梯式上升**——每个阶段有新的难度来源，同时玩家也获得了应对前一阶段难度的能力。

### 7.2 各阶段难度来源

| 阶段 | 主要难度来源 | 玩家能力 | 难度感受 |
|------|------------|---------|---------|
| 蛮荒期 | 不知道什么能吃什么不能吃、异兽未知 | 无 | 恐惧+好奇 |
| 定居期 | 科技负债开始显现、季节压力 | 基础生存能力 | 紧张+管理 |
| 文明期 | 灾变强度递增、生态管理复杂 | 多科技组合 | 挑战+策略 |
| 神话期 | 远征风险、传说事件链 | 高级科技+修仙 | 史诗+赌注 |

### 7.3 难度调节阀

以下机制可以作为难度调节阀，在测试中调整：

| 调节阀 | 调高难度 | 调低难度 |
|--------|---------|---------|
| 异兽攻击性 | 增加主动攻击范围 | 降低攻击频率 |
| 灾变频率 | 缩短灾变间隔 | 延长灾变间隔 |
| 食物腐败速度 | 加快腐败 | 减缓腐败 |
| 天赋变异概率 | 降低正向变异 | 提高正向变异 |
| 远征距离 | 增加资源点距离 | 减少距离 |
| 生态恢复速度 | 减慢恢复 | 加快恢复 |
| NPC帮助频率 | 减少NPC信息 | 增加NPC提示 |

## 八、各阶段激励重点

### 8.1 蛮荒期（0-5小时）

**玩家状态**：一无所知，随机降生于某个村落附近。

**激励重点**：好奇心驱动探索。

| 激励 | 实现 |
|------|------|
| 发现新事物 | 每次探索新区域都能遇到未见过的异兽/植物 |
| 图鉴解锁 | 频繁解锁图鉴槽位，提供即时正反馈 |
| 生存挑战 | 饥饿、口渴、寒冷迫使玩家学习基础生存 |
| NPC引导 | 村落NPC提供基础信息（如「那棵草叫祝余，听说吃了不饿」） |

**难度来源**：信息不对称。不知道蝮虫有毒、不知道九尾狐能吃人、不知道什么能吃什么不能吃。

**关键体验**：第一次吃祝余发现不饿的惊喜、第一次被蝮虫咬中毒的恐慌、第一次成功生火的成就感。

### 8.2 定居期（5-20小时）

**玩家状态**：已掌握基础生存，开始建造住所、发展科技。

**激励重点**：科技发展驱动建设。

| 激励 | 实现 |
|------|------|
| 科技解锁 | 每个科技节点解锁新配方/新交互 |
| 基地建设 | 从茅屋到木屋到石屋的视觉进步 |
| 季节挑战 | 第一个冬天的食物储备压力 |
| 繁衍后代 | 第一次生育后代，血脉延续 |

**难度来源**：科技负债开始显现。伐木导致森林减少、冶炼消耗大量燃料、过冬需要储备食物。

**关键体验**：第一个冬天差点饿死但挺过来了、第一次炼出铜刀的成就感、第一次看到后代出生的感动。

### 8.3 文明期（20-50小时）

**玩家状态**：多科技并行发展，村落初具规模，开始面对灾变。

**激励重点**：灾变应对驱动策略。

| 激励 | 实现 |
|------|------|
| 灾变预兆 | 识别预兆异兽，提前准备 |
| 生态管理 | 平衡资源开发与生态保护 |
| 跨科技组合 | 如「灌溉+育种」产出高产作物 |
| 修仙起步 | 引气感知灵气，开始修仙之路 |

**难度来源**：灾变强度与科技水平挂钩。生态反馈系统开始生效。科技管理复杂度上升。

**关键体验**：第一次成功预测洪水并提前迁移、第一次通过祭祀获得山神庇佑、第一次引气感知灵气的神秘体验。

### 8.4 神话期（50小时+）

**玩家状态**：科技树大部分解锁，开始远征和修仙。

**激励重点**：传说驱动冒险。

| 激励 | 实现 |
|------|------|
| 远征探索 | 前往未知的远方获取传说材料 |
| 修仙进阶 | 筑基、御物、御兽的修仙体验 |
| 神话事件链 | 触发鲧/羿/夸父等神话事件 |
| 飞升挑战 | 修仙满级后的飞升尝试 |

**难度来源**：远征的高风险、传说异兽的强大、飞升失败的永久死亡风险。

**关键体验**：第一次远征满载而归的狂喜、第一次御剑飞行的自由感、第一次面对传说异兽的敬畏、飞升成功/失败的终极时刻。

## 九、与现有系统的整合

### 9.1 与科技树的对接

本激励方案与附录2科技树系统的对接方式：

| 科技节点 | 激励方案中的角色 | 负债来源 |
|---------|----------------|---------|
| 采集→狩猎→生火 | 蛮荒期核心激励 | 食物链消耗 |
| 建造→制皮 | 定居期建设激励 | 木材消耗→生态反馈 |
| 草药学→烹药→解毒术 | 知识系统核心载体 | 误认/误配风险 |
| 外科手术（四液→推翻） | 「解锁即负债」典型 | 误诊风险→推翻旧理论 |
| 引气→筑基→飞升 | 神话期终极激励 | 灵气依赖+飞升失败风险 |
| 基础祭祀→山神祭祀→通灵→祈愿 | 灾变应对手段 | 祭品消耗+触怒风险 |
| 石器→木工→金属加工→机械 | 文明发展主线 | 生态反馈链 |
| 种植→灌溉→畜牧→育种→百谷 | 定居→文明过渡 | 土壤肥力管理 |

### 9.2 与物品系统的对接

附录1重要物品系统与本激励方案的对接：

| 物品类别 | 激励角色 | 设计要点 |
|---------|---------|---------|
| 食用类（祝余/鯥鱼/狌狌） | 蛮荒期生存激励 | 初始未知，需图鉴解锁 |
| 佩戴类（迷榖/鹿蜀角/旋龟壳） | 探索+传承激励 | 鹿蜀角是传承赌局核心 |
| 预兆类（长右/顒/毕方） | 灾变预警系统 | 出现即预告灾变 |
| 传说类（凤皇/精卫/刑天） | 神话期远征目标 | 地理独占，需远征 |
| 材料类（蝮虫毒牙/白䓘汁） | 知识解锁激励 | 用途未知，需图鉴解锁 |

### 9.3 与祭祀系统的对接

祭祀系统在本激励方案中承担**灾变应对**和**风险缓解**的角色：

| 祭祀等级 | 激励角色 | 风险 |
|---------|---------|------|
| 基础祭祀 | 蛮荒期心理安慰 | 祭品消耗 |
| 山神祭祀 | 定居期庇佑+灾变缓解 | 祭品错误触怒山神 |
| 通灵术 | 文明期信息获取 | 通灵失败被附身 |
| 祈愿术 | 灾变主动应对 | 祈愿被拒降好感 |

**关键设计**：祭祀不是「花钱买buff」，而是**用资源换不确定性**。你投入祭品，可能获得庇佑（风调雨顺），也可能无效（神灵不理你），甚至可能降灾（祭品错误触怒山神）。这种不确定性让祭祀始终有张力。

### 9.4 与修仙系统的对接

修仙系统在本激励方案中是**神话期的终极激励**：

| 修仙阶段 | 激励角色 | 难度来源 |
|---------|---------|---------|
| 引气 | 新玩法解锁（感知灵气） | 需在灵气地修炼，离开则停滞 |
| 筑基 | 修仙正式入门 | 需辅助物品，失败走火入魔 |
| 御物 | 新交互方式（操控物体） | 持续消耗灵力 |
| 御兽 | 新交互方式（驱使异兽） | 需驯兽前置，可能反噬 |
| 飞升 | 终极目标 | 失败=永久死亡，无传承 |

**关键设计**：修仙不是「数值变强」，而是**解锁新的游戏维度**。引气后可以感知灵气（新的信息层），御物后可以隔空操作（新的交互方式），御兽后可以驱使异兽（新的战斗/运输方式）。每个阶段都是质变，不是量变。

## 十、反模式：需要避免的设计陷阱

### 10.1 避免数值通胀

| 错误做法 | 正确做法 |
|---------|---------|
| 铜刀攻击力10，铁刀攻击力20，钢刀攻击力40 | 铜刀可砍普通异兽，铁刀可砍甲壳异兽，钢刀可砍传说异兽——质的区别不是量的区别 |
| 生命值从100涨到1000 | 生命值始终100，但解锁防具/闪避/格挡等新防御方式 |
| 经验值需求指数增长 | 无经验值系统，熟练度通过使用自然增长 |

### 10.2 避免虚假选择

| 错误做法 | 正确做法 |
|---------|---------|
| 六大科技树都能全解锁，只是顺序不同 | 科技树互斥节点——选了西医四液就不能选中医外科，选了修仙就不能学某些工匠科技 |
| 所有异兽都能用同一种策略击败 | 每种异兽有独特习性，需要不同应对方式 |
| 祭祀和炼丹效果重叠 | 祭祀管灾变/庇佑，炼丹管恢复/增益，功能不重叠 |

### 10.3 避免惩罚性难度

| 错误做法 | 正确做法 |
|---------|---------|
| 角色死亡丢失所有进度 | 知识/科技/图鉴永久保留，只丢失熟练度/物品 |
| 灾变随机发生无预警 | 灾变有预兆异兽预警，给玩家准备时间 |
| 远征失败直接Game Over | 远征失败只损失物资和角色（可传承），不影响血脉存续 |
| 误认草药直接死亡 | 误认草药中毒后有解毒时间，可被NPC救治 |

### 10.4 避免内容空转

| 错误做法 | 正确做法 |
|---------|---------|
| 图鉴只是装饰，解锁与否不影响玩法 | 图鉴解锁直接开启新交互选项 |
| 季节只是背景变化 | 季节影响食物/异兽/灾变，是生存策略的一部分 |
| NPC只提供对话 | NPC提供关键信息、交易、繁衍配偶、雇佣向导 |
| 远征只是走路 | 远征途中有随机事件、资源点、叙事内容 |

### 10.5 避免单一最优解

| 错误做法 | 正确做法 |
|---------|---------|
| 金属加工一定比石器强 | 金属加工有生态代价，石器在某些场景更优（无声狩猎） |
| 修仙一定比不修仙强 | 修仙需要灵气地绑定，限制了活动范围 |
| 定居一定比游牧强 | 定居有灾变风险，游牧可以规避灾变但无法积累基建 |
| 大村落一定比小村落强 | 大村落灾变强度更高，小村落更灵活 |

| 机制 | 原文依据 |
